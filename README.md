# 📌electronicPictureFrame
>전자액자

## ✔기능
<ul>
<li>저장소 접근 권한을 이용해 로컬 사진을 로드 할 수 있다.</li>
<li>추가한 사진들을 5초 간격으로 전환하여 보여줄 수 있다</li>
</ul>
